# Shahla's Sweet Corner (Duplicate file)

## What this is
A cozy café order builder in React. Users browse a dessert menu,
select items, write a personal note, and send a gift-like order
card to someone via email using EmailJS. The experience should
feel like sending a warm digital gift, not placing an online order.

## Aesthetic
Kawaii pixel art meets cozy bakery. The references are Chef RPG,
Stardew Valley, Harvest Town, and lofi café illustration style.
Think warm, soft, nostalgic — like a game you'd want to live inside.

Every screen should feel like a cozy game UI, not a website.
Tactile, warm, physical-feeling panels and cards. Nothing flat
or generic.

## Screen Flow
Landing → Menu → Write Note → Pay Animation → Send

## Screen Personalities
- Landing: warm and inviting, like arriving at a cozy café
- Menu: feels like browsing a beautiful café menu book,
  paginated 3 columns x 4 rows, 12 items per page
- Write Note: intimate, like writing on a little notepad


## Key Details
- EmailJS handles email delivery, no backend needed
- No prices or totals shown in email or on send screen
- The email should feel like receiving a gift, not a receipt
- EmailJS credentials stored as environment variables.
